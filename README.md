# 💼 Employee Payroll System

A terminal-based Employee Payroll System developed using **Python** and **Object-Oriented Programming (OOP)** concepts. This project is designed to manage employee records and automate salary calculations through a simple command-line interface.

## 🚀 Project Overview

The Employee Payroll System helps organizations manage employee payroll information efficiently. It allows users to add employee details, calculate salaries, and display payroll information directly from the terminal.

This project focuses on implementing core Python programming and OOP principles while simulating a real-world payroll management application.

## ✨ Features

✅ Add employee records

✅ Store employee information

✅ Calculate employee salaries

✅ Display payroll details

✅ Terminal-based user interaction

✅ Modular and reusable code structure

✅ Object-Oriented Programming implementation
